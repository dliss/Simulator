SourceGenerator¶

class SourceGenerator.SourceGenerator(rate=34, start=10000, end=22627000, distribution='no', print_locations='no')¶

provides a framework to simulate multiple source’s in the same run

SourceGenerator.generate_sources()¶

generates a specific number of sources based on the length of the simulation and the user-defined source rate

SourceGenerator.one_source()¶

runs the simulations for exactly one source.

SourceGenerator.find_emission_start()¶

randomly selects a start time for a transient’s flare

SourceGenerator.find_galactic_longitude()¶

randomly selects a galactic longitude for a source

SourceGenerator.find_galactic_lattitude()¶

randomly selects a galactic latitude for a source

SourceGenerator.print_locations()¶

prints the text for a region file to console.
