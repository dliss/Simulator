Variable Gamma-Ray Source Simulator Documentation
=====================================

 Welcome to the Variable Gamma-Ray Source Simulator! The Simulator is a package
 of python modules that can simulate the light curves of gamma-ray variable
 sources. This is the place to learn about the software.
